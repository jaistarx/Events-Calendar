# Events-Calendar

A desktop application made using Java Swing to create calendar events

## To Run

#### On Any Platform
Run from the parent directory(Events-Calendar)
```
cd calendar/target
```
```
java -jar starx-calender-1.0-SNAPSHOT.jar
```
#### On Linux

Run from the parent directory(Events-Calendar)
```
cd calendar/scripts
```
```
./calendarli.sh
```

#### On Windows
Run from the parent directory(Events-Calendar)
```
cd calendar/scripts
```
```
calendarwi.cmd
```
